# Blogging Platform API

A high-performance RESTful API built with **Rust** and the **Rocket** web framework for [roadmap.sh's Blogging Platform API project](https://roadmap.sh/projects/blogging-platform-api). This project focuses on type-safe endpoint handling, SQLite persistence, and structured data management, tested automatically in developing thanks to **Rust built-in testing**.
